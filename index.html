課程定位

以 實戰導向 為核心，掌握高品質 Prompt 設計與 AI 工具鏈，快速建立你的內容產線與自動化流程。

你將學到

  • 高命中率的 Prompt 結構：角色、目標、限制、評分規則
  • 文字 × 圖像 × 語音的多模態協作
  • 從範例到產出：把靈感變成作品

你將帶走

  • 可複用的提示詞範本庫
  • 你的第一版 AI 作品（含檔案）
  • 一套能持續優化的工作流程

課程大綱

第一部分 · 核心心法（45 分）

  • 生成式 AI 的能力邊界與常見誤區
  • Prompt 公式：目標 → 步驟 → 範例 → 評分標準
  • 如何讓 AI 保持一致：角色與風格記憶

第二部分 · 實作演練（90 分）

  • 商務稿件與簡報腳本快速成形
  • 社群貼文與短影音腳本生成
  • 圖片生成與版面應用（封面、縮圖）

第三部分 · 上線與自動化（45 分）

  • 將產出放到 GitHub Pages/Notion/雲端
  • 建立你的 Prompt 範本庫與命名規則
  • 日常迭代：如何快速 AB 測試與修正

延伸主題（選修）

  • 品牌語氣統一與多語系在地化
  • 資料導入與隱私風險治理概念

提示詞範本（可直接改字使用）

你是資深內容總監。請用「目標 → 步驟 → 範例 → 常見錯誤」四段式，
為我產出 3 個不同版本的貼文草稿；每個版本 80~120 字，
語氣：溫暖專業；平台：IG；加上 5 組主題標籤。
